## Instructions for Adding to Sanity

1. **First, create the Event for 2025** (if not already created):
   - Go to Sanity Studio → Events
   - Create or edit the 2025 event
   - Make sure it's set to "past" status

2. **Create Teams**:
   - Go to Sanity Studio → Teams
   - For each team listed above:
     - Create a new Team document
     - Set the name, category (EV or CV), and other details
     - Link the team to the 2025 event in "Registered Events"

3. **Create Results**:
   - Go to Sanity Studio → Results
   - For each result:
     - Select the Event (2025)
     - Select the Category (EV or CV)
     - Select the Subcategory (overall, engineering-design, etc.)
     - Select the Team
     - Enter Vehicle Number
     - Enter Position
     - Enter Points (Score)
     - Enter Penalties (if applicable)
     - Set Year to 2025

## Notes

- Teams with the same position should have the same position number
- Scores of 0 are valid results (team participated but didn't score)
- Penalties are only applicable to Overall CV results
- Vehicle numbers are important identifiers (E## for EV, C## for CV)
